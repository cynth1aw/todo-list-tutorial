What is SLAY110?

SLAY110 is a bi-annual hackathon event open to all COMP 110 students!

SLAY110 is an epic 12 hour event in which students can put their new-found coding skills to use! Students have the option to pursue either web development or game development in a beginner-friendly environment. No prior coding experience is necessary!

Get ready for 12 hours of coding and fun!
